## How to use

Clone the repository

```shell

```

Run yarn at the root of the workspace

```shell
cd ricecake-contracts
yarn
```

Testing contracts

```shell
npx hardhat node
npx hardhat test
```

Deploying to a testnet network

```shell
cp .env.example
npx hardhat run scripts/deployToRiceCake.js --network goerli
```
